        <p>This should be read in the context of layer 2, but is read first of all, out of its real context.</p>

</div>

<div class="div2">

        <p>Layer 2</p>

        <p>This should be read third, but is read second.</p>

</div>

        <p>This should be read first, but is read third.</p>

</div>

<div class="div4">

        <p>Layer 4</p>

        <p>This should be read second, but is read fourth.</p>

</div>

<div class="div5">

        <p>Layer 5</p>

        <p>This should be the last layer read on the page, but it isn't.</p>

</div>

<div class="div6">

        <p>Layer 6</p>
